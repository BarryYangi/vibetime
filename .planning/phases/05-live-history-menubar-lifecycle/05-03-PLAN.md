Refactor main process window code to include:

```typescript
let mainWindow: BrowserWindow | null = null
let isQuitting = false

function createMainWindow(): BrowserWindow
function showMainWindow(route?: string): void
function hideMainWindow(): void
function quitApp(): void
```

Rules:

- CLI mode remains before app identity/window initialization.
- Normal close prevents default and hides the window when `isQuitting === false`.
- `quitApp()` sets `isQuitting = true` and calls `app.quit()`.
- `app.on('before-quit')` sets `isQuitting = true`.
- `app.on('activate')` calls `showMainWindow()`.
- Open route uses last route when no route is passed.
